# 看板应用用户操作手册

## 1. 应用概述

看板应用是一个用于管理任务和工作流程的工具，通过可视化的方式展示任务状态和进度。本应用支持卡片的创建、编辑、删除和状态管理，帮助用户更有效地组织和跟踪工作。

## 2. 核心功能

### 2.1 查看看板

- **访问地址**: http://localhost:5175/
- **界面布局**: 应用界面包含四个状态列：待处理、进行中、已完成、已拒绝
- **卡片显示**: 每个卡片显示标题、内容和当前状态

### 2.2 创建卡片

1. 点击页面右上角的「Add Card」按钮
2. 在弹出的表单中填写：
   - **标题**: 卡片的标题（必填）
   - **内容**: 卡片的详细描述（必填）
   - **状态**: 卡片的初始状态（可选，默认为「待处理」）
3. 点击「Create Card」按钮完成创建
4. 新创建的卡片会出现在对应状态的列中

### 2.3 编辑卡片

1. **进入编辑模式**: 点击任意卡片进入编辑状态
2. **修改内容**: 
   - 编辑标题（文本输入框）
   - 编辑内容（文本域）
3. **保存修改**: 点击编辑界面中的「✓」按钮保存修改
4. **取消编辑**: 点击编辑界面中的「✖」按钮取消修改

### 2.4 删除卡片

1. **删除卡片**: 点击卡片右上角的「×」按钮
2. 卡片会立即从看板中移除

### 2.5 更改卡片状态（拖放操作）

1. **拖动卡片**: 点击并按住卡片，然后将其拖动到目标状态列
2. **释放卡片**: 在目标状态列中释放卡片
3. **状态更新**: 卡片会自动更新状态并保存到系统

## 3. 操作流程

### 3.1 基本工作流程

1. **创建任务**: 将新任务添加到「待处理」列
2. **开始工作**: 将任务从「待处理」拖放到「进行中」列
3. **完成任务**: 将任务从「进行中」拖放到「已完成」列
4. **拒绝任务**: 如果任务不需要执行，将其拖放到「已拒绝」列
5. **编辑任务**: 随时点击卡片进行内容修改
6. **删除任务**: 当任务不再需要时删除它

### 3.2 常见操作示例

#### 示例1: 添加新任务
1. 点击「Add Card」按钮
2. 填写标题「完成项目文档」
3. 填写内容「编写项目的技术文档和用户手册」
4. 选择状态「待处理」
5. 点击「Create Card」

#### 示例2: 开始处理任务
1. 找到「完成项目文档」卡片
2. 点击并拖动到「进行中」列
3. 释放鼠标

#### 示例3: 编辑任务内容
1. 点击「完成项目文档」卡片
2. 将内容修改为「编写项目的技术文档、用户手册和API文档」
3. 点击「✓」按钮保存修改

#### 示例4: 完成任务
1. 找到「完成项目文档」卡片
2. 点击并拖动到「已完成」列
3. 释放鼠标

## 4. 界面元素说明

### 4.1 顶部导航栏
- **应用标题**: 显示「Kanban Board」
- **Add Card按钮**: 用于创建新卡片

### 4.2 状态列
- **待处理**: 存放尚未开始的任务
- **进行中**: 存放正在处理的任务
- **已完成**: 存放已完成的任务
- **已拒绝**: 存放被拒绝或不需要执行的任务

### 4.3 卡片元素
- **标题**: 卡片的标题，显示在卡片顶部
- **内容**: 卡片的详细描述，显示在标题下方
- **状态**: 卡片当前的状态，显示在卡片底部
- **删除按钮**: 位于卡片右上角，用于删除卡片

### 4.4 编辑界面
- **标题输入框**: 用于编辑卡片标题
- **内容文本域**: 用于编辑卡片内容
- **保存按钮**: 用于保存修改
- **取消按钮**: 用于取消修改
- **状态显示**: 显示当前卡片状态（不可编辑，需通过拖放更改）

## 5. 快捷键

| 操作 | 快捷键 | 说明 |
|------|--------|------|
| 创建卡片 | 无 | 点击「Add Card」按钮 |
| 编辑卡片 | 无 | 点击卡片进入编辑模式 |
| 保存修改 | Ctrl+Enter | 在编辑模式下保存修改 |
| 取消编辑 | Esc | 在编辑模式下取消修改 |
| 删除卡片 | 无 | 点击卡片上的删除按钮 |

## 6. 注意事项

1. **必填字段**: 创建卡片时，标题和内容为必填字段
2. **状态变更**: 状态变更只能通过拖放操作完成，不能在编辑界面修改
3. **数据保存**: 所有操作都会实时保存到服务器
4. **错误处理**: 如果操作失败，会在控制台显示错误信息
5. **浏览器兼容性**: 建议使用现代浏览器（Chrome、Firefox、Edge）访问

## 7. 故障排除

### 7.1 常见问题

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| 卡片创建失败 | 网络连接问题或服务器错误 | 检查网络连接，刷新页面后重试 |
| 拖放操作无效 | 浏览器不支持拖放功能 | 使用现代浏览器，如Chrome或Firefox |
| 编辑保存失败 | 网络连接问题或服务器错误 | 检查网络连接，刷新页面后重试 |
| 卡片不显示 | 网络连接问题或服务器错误 | 检查网络连接，刷新页面 |

### 7.2 技术支持

如果遇到无法解决的问题，请检查浏览器控制台的错误信息，并联系技术支持人员。
