# 看板系统 v0.5 UI 设计文档

## 1. 设计概述

本设计文档基于v0.5版本的产品需求文档，详细描述了看板系统的UI设计方案，包括布局、样式和状态颜色方案。

## 2. 布局设计

### 2.1 整体布局

- **布局容器**：创建一个宽度为100vw的布局组件
- **看板居中**：kanban-board元素在布局容器中居中显示
- **响应式设计**：适配不同屏幕尺寸

### 2.2 高度设计

- **kanban-board高度**：固定高度，占据整个浏览器高度
- **status-column高度**：固定高度，当卡片数量超过容器高度时，显示垂直滚动条

## 3. 组件设计

### 3.1 按钮设计

- **删除按钮**：
  - 边缘：圆形
  - 尺寸：适中，便于点击
  - 颜色：红色，醒目提示

### 3.2 状态Tag设计

| 状态 | 颜色 | 描述 |
|------|------|------|
| 待处理 | #3498db | 蓝色，代表任务等待处理 |
| 进行中 | #f39c12 | 黄色，代表任务正在进行 |
| 已完成 | #27ae60 | 绿色，代表任务已完成 |
| 已拒绝 | #e74c3c | 红色，代表任务已拒绝 |

### 3.3 卡片设计

- **卡片样式**：轻微阴影，悬停效果
- **内容布局**：清晰的标题和描述
- **交互**：支持拖拽操作

## 4. 样式规范

### 4.1 颜色系统

- **主色调**：#3498db（蓝色）
- **辅助色**：#2ecc71（绿色）
- **警告色**：#f39c12（黄色）
- **错误色**：#e74c3c（红色）
- **中性色**：#ecf0f1（浅灰）、#95a5a6（中灰）、#34495e（深灰）

### 4.2 字体系统

- **标题**：16px，加粗
- **正文**：14px，常规
- **辅助文字**：12px，常规

### 4.3 间距系统

- **卡片间距**：8px
- **列间距**：16px
- **容器内边距**：16px

## 5. 交互设计

- **拖拽操作**：平滑的拖拽效果
- **滚动效果**：流畅的垂直滚动
- **按钮点击**：轻微的点击反馈

## 6. 实现注意事项

- 使用CSS Flexbox或Grid实现居中布局
- 使用CSS height和overflow属性实现固定高度和滚动效果
- 使用CSS border-radius属性实现圆形按钮
- 使用CSS类或内联样式实现状态颜色
