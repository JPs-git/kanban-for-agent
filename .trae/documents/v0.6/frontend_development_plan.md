## 1. 开发目标

1. 重复添加卡片时模态框状态未清除
2. 点击卡片修改信息时模态框未自动填充

### 2.2 技术实现

#### 2.2.1 修复添加卡片模态框

- 文件：`frontend/src/components/AddCardModal.tsx`
- 实现：添加 `useEffect` 钩子，当 `isOpen` 状态变化为 `true` 时，重置表单状态
- 代码示例：
  ```typescript
  useEffect(() => {
    if (isOpen) {
      setNewCard({
        title: '',
        content: '',
        status: CardStatus.TODO,
        assignee: '',
        assigneeName: '未分配',
      });
    }
  }, [isOpen]);
  ```

#### 2.2.2 修复编辑卡片模态框

- 文件：`frontend/src/components/CardEditModal.tsx`
- 实现：添加 `useEffect` 钩子，当 `card` 属性变化时，更新表单状态
- 代码示例：
  ```typescript
  useEffect(() => {
    if (card) {
      setEditTitle(card.title || '');
      setEditContent(card.content || '');
      setEditStatus(card.status || CardStatus.TODO);
      setEditAssignee(card.assignee || '');
    }
  }, [card]);
  ```
